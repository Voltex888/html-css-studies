Oque é CSS?

CSS (Cascading Style Sheets) é uma líguagem de estilo que serve para mudar a aparência de páginas web

com o HTML Criamosa estrutura dessas páginas.
Mas mudar a aparência delas com HTML é ímpossivel.
Para isso ultilizamos o CSS

HTML - Serve para estruturar uma página web

CSS - Serve para estilizar uma página web
